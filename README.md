# Simulador de Orçamento Freelancer de TI

## Descrição

Este projeto é um simulador de orçamento freelancer na área de Tecnologia da Informação.  
A aplicação permite que um cliente selecione serviços de TI e visualize o valor total do orçamento de forma dinâmica na tela.

O projeto foi desenvolvido como atividade prática da disciplina de Front-end.

## Objetivo

Desenvolver uma aplicação web simples utilizando HTML, CSS e JavaScript para simular a contratação de serviços de TI e calcular automaticamente o valor total do orçamento.

## Serviços Disponíveis

- Landing Page
- Site Institucional
- Hospedagem
- Banco de Dados
- Integração de API
- Manutenção Mensal

## Tecnologias Utilizadas

- HTML
- CSS
- JavaScript

## Funcionalidades

- Seleção de serviços
- Cálculo automático do valor total
- Atualização dinâmica do orçamento
- Interface simples e organizada

## Conceitos Aplicados

- Estrutura HTML
- Estilização com CSS
- Manipulação do DOM
- Variáveis (let e const)
- Eventos (onchange)
- parseFloat()
- Arrow Function

## Precificação

Os valores dos serviços foram definidos com base em pesquisas de mercado e plataformas de freelancers, considerando médias praticadas na área de desenvolvimento web e serviços de TI.

## Como Executar

1. Baixar o projeto
2. Abrir a pasta
3. Executar o arquivo index.html no navegador
